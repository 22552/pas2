# pas2
pas2は、フリーでオープンソースなウェブフレームワークのプロジェクトです！

非常に軽量で、<a href="https://github.com/yayoiP/pyappscript">pyappscript</a>というフレームワークからのフォークです。

今後、WSGIではなく、ASGIに対する対応を強化するため、ASGI機能に焦点をあて、重大な脆弱性以外は対策せず、廃止予定です。FlaskのBlueprintとして当フレームワークのWSGI機能を実施してください。WSGIは削除されます

このプロジェクトは発展途上であるため、十分な機能を利用するなら、脆弱な<a href="https://github.com/yayoiP/pyappscript">pyappscript</a>を利用してください。
