# pas2
pas2は、軽量・シンプル設計を重視したPython製のASGIウェブフレームワークです

かつてのWSGI時代の設計を土台にしつつ、現代的な非同期モデル（ASGI）へ再構築しています。

## 目指しているもの
- ASGIファースト設計
- 依存を抑えた軽量構成
- 直感的なルーティング
- シンプルなrequestオブジェクト
- WebSocket対応
- 小さく理解しやすいコア

コアをベースに、Bottleを意識したラッパーを作成する予定です！
  
## 将来方針
pas2は、WSGIサポートを段階的に廃止し、ASGI専用フレームワークへ移行します。

互換レイヤーとして、必要に応じてFlask Blueprint経由で利用可能ですが、コアはASGIに集中します

## 目標
「ASGI時代の軽量フレームワーク」という立場の確立
書き味を保ちながら内部を進化させる設計
HTTP/3時代を見据えた基盤の整備
ステータス
このプロジェクトは現在も開発中です。
破壊的変更が入る可能性があります。
